# HOSPITAL_EMERGENCY_ROOM_ANALYSIS_DASHBOARD
I have developed an interactive Hospital Emergency Room dashboard to analyze and monitor patient flow, average wait time, and satisfaction scores. I have also included filters for year and months along with visual insights on admissions, age, groups, gender distribution, and department referrals to support data driven decisions.
